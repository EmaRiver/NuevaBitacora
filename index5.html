CODIGOS JAVA

Funciones

¿Qué son las funciones?

Las funciones son bloques de código reutilizables que realizan una tarea específica. Permiten organizar el código y evitar repetir las mismas instrucciones.

¿Para qué sirven?

- Reutilizar código
- Organizar y estructurar programas
- Facilitar el mantenimiento
- Dividir problemas complejos en partes más pequeñas

Funciones que Retornan Valores

¿Qué son?

Funciones que devuelven un resultado usando la palabra clave return. Este valor puede almacenarse en variables o usarse directamente.

¿Para qué sirven?

- Realizar cálculos y devolver resultados
- Procesar datos y retornar información
- Crear funciones utilitarias reutilizables
- Encapsular lógica compleja

Llamada a Función

¿Qué es?

Es la acción de ejecutar una función escribiendo su nombre seguido de paréntesis. Si la función requiere parámetros, se pasan dentro de los paréntesis.

¿Para qué sirve?

- Ejecutar el código dentro de una función
- Pasar datos a las funciones mediante parámetros
- Obtener resultados de funciones que retornan valores
- Controlar el flujo del programa

Console.log()

¿Qué es?

Es una función que imprime mensajes en la consola del navegador. Es una herramienta fundamental para depurar código y entender qué está pasando en el programa.

¿Para qué sirve?

- Depurar errores en el código
- Ver valores de variables en tiempo real
- Seguir el flujo de ejecución del programa
- Mostrar mensajes informativos
- Probar funciones rápidamente

GetElementById()

¿Qué es?

Es un método del DOM (Document Object Model) que permite seleccionar un elemento HTML específico usando su atributo ID.

¿Para qué sirve?

- Acceder a elementos específicos de la página
- Modificar contenido HTML dinámicamente
- Cambiar estilos CSS desde JavaScript
- Agregar o quitar clases CSS
- Manipular atributos de elementos

AddEventListener()

¿Qué es?

Es un método que permite "escuchar" eventos específicos en elementos HTML y ejecutar código cuando ocurren esos eventos.

¿Para qué sirve?

- Responder a interacciones del usuario (clics, teclas)
- Crear interfaces interactivas
- Validar formularios en tiempo real
- Crear efectos visuales dinámicos
- Manejar diferentes tipos de eventos del navegador
